# Data Analysis Application

## Description

The Data Analysis Application is a simple graphical user interface (GUI) tool created in Python for analyzing CSV data files. This application allows you to easily load data, view statistics, create plots, and perform basic data manipulation tasks such as adding, updating, and deleting rows.

## Features

- **Load Data**: Load data from a CSV file.
- **Show Plot**: Display a plot of the data in a new window.
- **Save Statistics**: Save the data statistics to a text file.
- **Add Row**: Add a new row to the data.
- **Update Row**: Update an existing row in the data.
- **Delete Row**: Delete a row from the data.

## Requirements

- Python 3.7+
- Python libraries:
  - `customtkinter`
  - `tkinter` (included with Python)
  - `pandas`
  - `matplotlib`
